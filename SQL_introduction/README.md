# 🗄 SQL Introduction

Ce projet introduit les bases du langage **SQL** avec **MySQL**.

Les scripts permettent de manipuler :

* les bases de données
* les tables
* les insertions
* les sélections
* les agrégations

---

# 🎯 Objectifs

* comprendre la syntaxe SQL
* créer et supprimer des bases de données
* créer des tables
* manipuler les données
