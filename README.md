# 🐍 Higher Level Programming - Holberton School

Ce repository regroupe les projets réalisés dans le cadre du module **Higher Level Programming** à Holberton School.

L'objectif de ces projets est de découvrir et maîtriser le **langage Python**, la **programmation orientée objet**, ainsi que la **manipulation des bases de données et des API**.

---

# 🎯 Objectifs pédagogiques

À travers ces projets, j’ai appris à :

* écrire des scripts en Python
* comprendre les bases de la programmation orientée objet (POO)
* manipuler les exceptions
* interagir avec des bases de données
* utiliser les modules et packages Python
* écrire du code propre et structuré
