# 🐍 Python - Classes

Ce projet introduit les **classes et objets en Python**.

Les exercices permettent de comprendre :

* la création de classes
* les attributs
* les méthodes
* l'encapsulation

---

# 🎯 Objectifs

* comprendre la programmation orientée objet
* créer des classes Python
* manipuler les attributs
* utiliser des méthodes
