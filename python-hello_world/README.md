# 🐍 Python - Hello World

Ce projet introduit les bases du langage **Python** dans le cadre du module **Higher Level Programming**.

Les exercices permettent de découvrir :

* l'interpréteur Python
* l'exécution de scripts
* la manipulation de chaînes
* l'affichage de variables

---

# 🎯 Objectifs

* apprendre la syntaxe de base de Python
* afficher du texte et des variables
* manipuler les chaînes de caractères
* comprendre l'exécution de scripts Python
