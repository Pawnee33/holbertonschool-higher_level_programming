# 🐍 Python - If / Else / Loops / Functions

Ce projet permet de découvrir les structures de contrôle en Python.

Les exercices couvrent :

* conditions
* boucles
* fonctions
* manipulation d'entiers

---

# 🎯 Objectifs

* utiliser `if`, `else`
* utiliser les boucles `for`
* créer des fonctions
* manipuler les entiers
