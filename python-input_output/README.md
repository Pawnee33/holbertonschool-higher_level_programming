# 🐍 Python - Input / Output

Ce projet introduit la gestion des **fichiers et des entrées/sorties en Python**.

Les exercices permettent de manipuler :

* la lecture de fichiers
* l’écriture de fichiers
* la sérialisation JSON
* la manipulation de données

---

# 🎯 Objectifs

* lire et écrire dans des fichiers
* convertir des objets Python en JSON
* charger des données JSON
* manipuler des classes et leurs attributs
